Detailed Description

Animate along a set of points that need to be in the format: controlPoint, point1, point2....

pointLast, endControlPoint

Move a GameObject to a certain location

@constructor

Parameters
{Vector3Array} pts A set of points that define the points the path will pass through (starting with starting control point, and ending with a control point)
Note: The first and last item just define the angle of the end points, they are not actually used in the spline path itself. If you do not care about the angle you can jus set the first two items and last two items as the same value.
